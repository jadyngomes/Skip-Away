# Skip Away

Skip Away is a game coded on Visual Studio Code using vanilla javascript, HTML and CSS. Users are required to 
bounce of the incoming platforms and keep the ball in the air. Once the ball touches the bottom of the grid,
the game ends. 


## Loading the game

Open the 'SKIP AWAY.html' to open the homepage
Select an theme by clicking on the pictures displayed
Clicking START loads the game
Clicking HOW TO PLAY shows the instructions of the game
To go back to the homepage, press the back arrow icon on your web browser


## Usage/Movement

Right arrow key on the key board moves the ball Right
Left arrow key on the key board moves the ball to the Left


## Files
Ensure all files are in the same folder. Copy the entiure folder if necessary and not the 'SKIP AWAY.html' alone
